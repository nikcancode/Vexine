# Vexine - Personal Health & Fitness Guide

A comprehensive BMI calculator and health guide application built with Python and Tkinter.

## Features

- **BMI Calculation** - Accurate body mass index calculation with WHO standards
- **Personalized Recommendations** - Custom nutrition and exercise tips based on:
  - Current BMI and body type
  - Fitness level (beginner/intermediate/advanced)
  - Personal goals (weight loss, muscle gain, maintenance)
  - Desired body type
- **Smart Nutrition Tips** - Tailored meal planning and dietary advice
- **Exercise Programs** - Workout recommendations for all fitness levels
- **Lifestyle Guidance** - Holistic health and wellness tips
- **Professional UI** - Clean, modern interface with fullscreen support

## Requirements

- Python 3.6 or higher (tested on Python 3.14.0)
- Tkinter (comes pre-installed with Python)

## Installation

1. Clone this repository:
git clone https://github.com/nikcancode/vexine.git
cd vexine
2. Verify Python installation:
python --version
3. Test Tkinter:
python -m tkinter

## Usage
Run the application:
python vexine.py

### Keyboard Shortcuts
- **ESC** - Exit fullscreen mode
- **F11** - Toggle fullscreen

## How It Works

1. Enter your basic information (age, gender, height, weight)
2. Select your current body type and fitness level
3. Choose your desired body type and primary goal
4. Click "Calculate BMI & Get Recommendations"
5. View your personalized health and fitness plan
